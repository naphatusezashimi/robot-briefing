# คำสั่งเริ่มต้นสำหรับ Claude Code

ไฟล์นี้คือ "บทพูด" ที่ก๊อปไปวางใน Claude Code ได้เลย ทำตามลำดับ

---

## ขั้นที่ 0: ติดตั้ง Claude Code (ทำครั้งเดียว)

ต้องมีบัญชี Claude (เช่น Pro/Max) หรือ API key และต้องเชื่อมอินเทอร์เน็ต
วิธีติดตั้งที่แนะนำตอนนี้คือ native installer ซึ่งไม่ต้องลง Node.js
เปิด terminal แล้วรันคำสั่งตามระบบปฏิบัติการ:

- **macOS / Linux:** `curl -fsSL https://claude.ai/install.sh | bash`
- **Windows (PowerShell):** `irm https://claude.ai/install.ps1 | iex`

(ถ้าถนัด npm และมี Node.js 18+ อยู่แล้ว ใช้
`npm install -g @anthropic-ai/claude-code` ก็ได้ แต่อย่าใส่ sudo เพราะจะติด
ปัญหาสิทธิ์)

ติดตั้งเสร็จ พิมพ์ `claude --version` เช็กว่าใช้ได้ จากนั้น `cd` เข้าไปใน
โฟลเดอร์โปรเจกต์นี้ (โฟลเดอร์ที่มีไฟล์ BRIEF.md อยู่) แล้วพิมพ์ `claude`
เพื่อเปิด Claude Code

> วิธีติดตั้งล่าสุดดูได้ที่เอกสารทางการ: https://code.claude.com/docs/en/setup

---

## ขั้นที่ 1: ให้มันอ่านบรีฟก่อน

ก๊อปข้อความนี้วางเป็นคำสั่งแรก:

```
อ่านไฟล์ BRIEF.md, college_data.example.json และ robot_main_reference.py
ในโฟลเดอร์นี้ทั้งหมดก่อน แล้วสรุปให้ฟังว่าเข้าใจโปรเจกต์ว่าอย่างไร
และจะเริ่มเฟส 1 อย่างไร ยังไม่ต้องเขียนโค้ด รอให้ฉันยืนยันก่อน
```

---

## ขั้นที่ 2: เริ่มสร้างเฟส 1 (โหมดจำลอง)

พอมันสรุปเข้าใจตรงกันแล้ว ก๊อปข้อความนี้:

```
เริ่มเฟส 1 ตาม BRIEF.md ได้เลย สร้างโครงไฟล์ทั้งหมด ทำ database.py,
ai.py (อย่าลืม prompt กันตอบมั่ว), และ simulator.py ที่ให้ฉันพิมพ์
คำถามแล้วได้คำตอบจาก AI บนคอมโดยไม่ต้องมีอุปกรณ์จริง
ใช้ college_data.example.json เป็นฐานข้อมูลตั้งต้น
ทำ requirements.txt และ README วิธีรันด้วย
```

---

## ขั้นที่ 3: ทดสอบ

ทำตาม README ที่มันสร้าง ติดตั้งไลบรารีและตั้งค่า API key แล้วรัน
simulator พิมพ์คำถามทดสอบ เช่น:
- "วิทยาลัยเปิดสอนสาขาอะไรบ้าง"
- "รับสมัครเมื่อไหร่"
- "ค่าเทอม ปวช. เท่าไหร่"
- ลองถามเรื่องที่ไม่มีในฐานข้อมูล เพื่อเช็กว่ามันตอบว่า "ไม่ทราบ"
  จริงไหม (ห้ามมั่ว)

---

## ขั้นที่ 4: ไปเฟสถัดไป

พอเฟส 1 ใช้ได้แล้ว ค่อยบอกมันว่า:

```
เฟส 1 ใช้งานได้แล้ว เริ่มเฟส 2 ตาม BRIEF.md (เพิ่มส่วนเสียง STT/TTS
แบบยังจำลองได้)
```

---

## เคล็ดลับการใช้ Claude Code
- ทำทีละเฟส อย่ารีบ ทดสอบให้ผ่านก่อนไปต่อ
- ถ้ามันทำอะไรไม่ตรงใจ บอกแก้ได้ตรงๆ มันแก้ไฟล์ให้เอง
- ถ้าติดปัญหาออกแบบใหญ่ๆ หรืออยากปรึกษาทิศทาง กลับมาคุยกับ
  Claude ในแอป/เว็บได้ แล้วเอาผลไปให้ Claude Code ทำต่อ
