# 🤖 หุ่นยนต์แนะนำข้อมูลวิทยาลัยเทคนิคท่าหลวงซิเมนต์ไทยอนุสรณ์

หุ่นยนต์ **"กล่องพูดได้"** สำหรับตั้งที่วิทยาลัย ให้ผู้มาติดต่อ **พูดถามข้อมูลแล้วได้คำตอบเป็นเสียงภาษาไทย** — เรื่องหลักสูตร การรับสมัคร การติดต่อ และอื่นๆ

> 🎯 **หัวใจของระบบ:** ตอบจากฐานข้อมูลวิทยาลัยเท่านั้น **ไม่ตอบมั่ว** — ถ้าไม่มีข้อมูล จะบอกให้ติดต่อเจ้าหน้าที่ (โทร 0-3628-1295)

## ✨ ความสามารถ

- 🎤 **พูดถาม–พูดตอบ** ภาษาไทย (Web Speech API ของเบราว์เซอร์ — ฟรี)
- ⌨️ **พิมพ์ถามได้** เป็นตัวสำรอง (เผื่อคนไม่อยากพูด)
- 😊 **จอใบหน้าแสดงอารมณ์** (รอ / ฟัง / คิด / ตอบ / ผิดพลาด) + จอแชทแสดงบทสนทนา
- 🧠 ใช้ AI (**Gemini** ฟรีเป็นค่าเริ่มต้น / สลับเป็น **Claude** ได้) ตอบโดยอ้างอิง `college_data.json` เท่านั้น

## 🚀 เริ่มใช้งานเร็ว

```bash
cd robot
pip install -r requirements.txt
```

ใส่ `GEMINI_API_KEY` ในไฟล์ `robot/.env` แล้ว **ดับเบิลคลิก `robot/web/run_web.bat`**
→ เปิด Chrome ที่ `http://127.0.0.1:5000/screen` (จอแชท) และ `/face` (จอใบหน้า)

📖 รายละเอียดการติดตั้ง / สลับ AI / **วิธีย้ายไปโชว์บนเครื่องอื่น (โน้ตบุ๊ก)**: ดู **[robot/README.md](robot/README.md)**

## 📁 โครงสร้างโปรเจกต์

```
robot/                โค้ดหลัก
  ai.py, database.py  สมอง: ถาม AI + กฎกันตอบมั่ว + โหลดฐานข้อมูล
  college_data.json   ฐานข้อมูลวิทยาลัย (หุ่นตอบจากไฟล์นี้เท่านั้น)
  web/                เว็บ "กล่องพูดได้": server.py (Flask) + static/ (จอใบหน้า + จอแชท)
  simulator.py        โหมดจำลองแบบพิมพ์ใน terminal
mockups/              แบบหน้าจอ (ดีไซน์ UI)
docs/superpowers/     สเปก + แผนการพัฒนา
```

## 📌 สถานะ

| เฟส | สิ่งที่ทำ | สถานะ |
|-----|---------|-------|
| เว็บกล่องพูดได้ | พูด/พิมพ์ + จอใบหน้า + กันตอบมั่ว (ทดสอบบนคอม) | ✅ เสร็จ |
| ฮาร์ดแวร์ | ประกอบกล่อง + 2 จอจริง + ไฟ LED บน Raspberry Pi | ⏳ ถัดไป |
