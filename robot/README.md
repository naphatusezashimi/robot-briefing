# หุ่นยนต์แนะนำข้อมูลวิทยาลัยเทคนิคท่าหลวง

## ติดตั้ง

```bash
cd robot
pip install -r requirements.txt
```

## ตั้งค่า API Key

```bash
cp .env.example .env
```

จากนั้นเลือกเจ้า AI ใน `config.py` ที่บรรทัด `AI_PROVIDER`:

- **`"gemini"` (ค่าเริ่มต้น, ฟรี)** — ขอ key ฟรีที่ https://aistudio.google.com/apikey แล้วใส่ `GEMINI_API_KEY` ใน `.env`
- **`"claude"`** — ใส่ `ANTHROPIC_API_KEY` ใน `.env` (ต้องมีเครดิตในบัญชี Anthropic)

สลับระหว่างสองเจ้าได้โดยแก้ `AI_PROVIDER` บรรทัดเดียว

## รันโหมดจำลอง (เฟส 1)

วิธีง่ายสุดบน Windows: **ดับเบิลคลิกไฟล์ `run.bat`** (ตั้ง UTF-8 + เรียก Python ให้อัตโนมัติ)

หรือรันเองผ่าน terminal:

```bash
python simulator.py
```

พิมพ์คำถามเกี่ยวกับวิทยาลัย กด Enter รับคำตอบ พิมพ์ `q` เพื่อออก

> หมายเหตุ: ถ้าพิมพ์ภาษาไทยแล้วเพี้ยน ให้ใช้ `run.bat` หรือสั่ง `chcp 65001` ก่อน และตั้ง `set PYTHONUTF8=1`

## รันเวอร์ชัน "กล่องพูดได้" (เว็บ — พูด/พิมพ์ได้)

ติดตั้งเพิ่ม (ครั้งแรกครั้งเดียว): `pip install -r requirements.txt`

**วิธีง่ายสุด:** ดับเบิลคลิก `web/run_web.bat`
แล้วเปิดเบราว์เซอร์ (Chrome/Edge):
- จอเนื้อหา/แชท: http://127.0.0.1:5000/screen
- จอใบหน้า: http://127.0.0.1:5000/face

กดปุ่มไมค์เพื่อพูด หรือพิมพ์คำถามก็ได้ หุ่นจะตอบด้วยเสียงไทย + แสดงบนจอ
(ภาษาเสียงใช้ Web Speech API ของเบราว์เซอร์ — รองรับดีใน Chrome/Edge)

## ย้ายไปรัน/โชว์บนเครื่องอื่น (เช่น โน้ตบุ๊ก)

> เครื่องที่รันจะกลายเป็น "ตัวหุ่น" เลย — รันบนเครื่องตัวเอง (localhost) เสียงไมค์ใช้ได้เต็ม

**ตั้งค่าครั้งเดียว:**

1. **ติดตั้ง Python 3** จาก https://www.python.org/downloads/ → ตอนติดตั้ง **ติ๊ก "Add Python to PATH"** (สำคัญ เลี่ยงปัญหาหา Python ไม่เจอ)
2. **เอาโค้ดไปลงเครื่องใหม่** เลือกทางใดทางหนึ่ง:
   - ก๊อปโฟลเดอร์ทั้งอันผ่าน USB/OneDrive → ได้ไฟล์ `.env` (คีย์) ติดไปด้วย ไม่ต้องตั้งใหม่
   - `git clone https://github.com/naphatusezashimi/robot-briefing` → สะอาดกว่า แต่ `.env` ไม่ติดมา (git ไม่เก็บคีย์) ต้องสร้างเอง
3. ที่โฟลเดอร์ `robot` รัน `pip install -r requirements.txt`
4. เช็กว่ามีไฟล์ `robot/.env` ที่มี `GEMINI_API_KEY=...` (ถ้า clone มา ให้ `cp .env.example .env` แล้วใส่คีย์)
5. ดับเบิลคลิก `web/run_web.bat` → เปิด Chrome ที่ http://127.0.0.1:5000/screen

> `run_web.bat` หา Python ให้อัตโนมัติ (ลอง path เครื่องเดิมก่อน ถ้าไม่เจอใช้ `python` ใน PATH) จึงรันได้ทุกเครื่องที่ลง Python แล้ว

**เช็กลิสต์ก่อนวันโชว์:**

- ⭐ ลองรันบนเครื่องที่จะใช้โชว์ให้ผ่าน **ก่อนวันจริงอย่างน้อย 1 วัน** (อย่าให้ครั้งแรกคือตอนอยู่หน้ากรรมการ)
- ใช้ **Chrome** แล้วกด **"อนุญาต" ไมโครโฟน** ไว้ล่วงหน้า
- ต้องมี **อินเทอร์เน็ต** (ทั้งเสียงเบราว์เซอร์และ Gemini ใช้เน็ต) — เตรียม hotspot มือถือสำรองไว้
- เสียบปลั๊ก ปิด sleep/screensaver, เช็กไมค์+ลำโพงดังพอ

## อัปเดตข้อมูลวิทยาลัย

แก้ไขไฟล์ `college_data.json` — หุ่นจะตอบจากข้อมูลในไฟล์นี้เท่านั้น

## เปลี่ยนโหมด

แก้ `config.py` บรรทัดเดียว:

```python
SIMULATOR_MODE = True   # จำลองบนคอม
SIMULATOR_MODE = False  # ฮาร์ดแวร์จริงบน Pi (เฟส 3)
```

| เฟส | สิ่งที่ทำ | สถานะ |
|-----|---------|-------|
| 1 | โครง + โหมดจำลอง (ไฟล์นี้) | ✅ เสร็จ |
| 2 | เพิ่ม Google STT + TTS | ⏳ รอ |
| 3 | ต่อฮาร์ดแวร์จริงบน Pi | ⏳ รอ |
| 4 | Log + สถิติ | ⏳ รอ |
